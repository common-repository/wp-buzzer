WP Buzzer Plugin allows you to add a post/page or blog to your Google Buzz.

== Description ==

[WP Buzzer](http://artiko.net.pl/projekty/wordpress/wp-buzzer/) Plugin allows you to add a post/page/blog to Google Buzz. It displays a Google Buzz button after each post and page. It also features a widget that you can put in your sidebar.
You can manage it's visibility on the plugin options page.
Features:

* Display a Google Buzz button after each post and page
* Widget with Google Buzz button

You can see a demo on my website : [artiko.net.pl](http://artiko.net.pl/)

== Installation ==

    * Download it 
    * Extract all files from the zip archive
    * Copy the wp-buzzer folder to wp-content/plugins
    * Activate the plugin through the "Plugins" menu in wordpress
    * Add WP Buzzer Widget to your sidebar
    * That's all !
